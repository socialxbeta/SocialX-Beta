'SocialX' este o aplicație pentru android ce poate fi folosită pentru distribuirea unui continut, fie ca este vorba despre continut de tip text, imagine sau chiar ambele. Aplicatia face parte din categoria de programe de tip socializare (eng. Social). SocialX are la baza urmatoarele limbaje de programare: C++, eXtensible Markup Language, HTML, Java, Java Script, Kotlin, NoSQL, Python si SQL. In principal, aplicatia este creată cu ajutorul mediului de dezvoltare Android Studio, acesta folosind limbaje de programare precum Java, eXtensible Markup Language si Kotlin, dar si pogramarea orientata pe obiecte. Printr-o afiliatie dintre Google LTD. si SocialX, baza de date este situata in serverul principal al Google-lului din Los Angeles, Statele Unite ale Americii (eng. Los Angeles, US). Pentru crearea bazei de date au fost folosite limbajele de interogare precum SQL si NoSQL, iar ca platforma de administrare a bazei de date este folosita platforma Firebase, o platforma ce apartine Google-ului din 2011. Cum aceasta aplicatie este destinata publicului larg si stocheaza diferite date despre utilizatori, au fost folosite limbajele de programare 'Python' alaturi de 'Java' pentru o criptare sporita a datelor stocate on-line. Cum multi oameni folosesc in ziua de astazi filtrele pentru a isi imbunatatii pozele sau in unele cazuri eliminarea elementelor neplacute, am creat cu ajutorul limbajului de programare C++ un mediu virtual in care se pot indeplini aceste cerinte si in cadrul aplicatiei SocialX. Pentru interactionarea dintre baza de date si aplicatie, pe langa limbajele de interogare si de criptare, au fost folosite si functii care au rolul de a sincroniza baza de date si in acelasi timp de a extrage datele utile despre utilizatori in conformitate cu 'Termenii si Conditiile de Utilizare' si cu 'Politica de Confidentialitate'; functiile au fost create folosind limbajul Java Script. HTML a fost utilizat in aceasta aplicatie pentru a customiza fonturile folosite in campurile de tip text.
